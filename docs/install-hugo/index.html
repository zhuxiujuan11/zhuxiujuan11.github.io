Install Hugo

Prerequisites 

Before you begin building websites, you must:

  1. Install Hugo (the extended version)
  2. Install Git

Some Hugo themes use Hugo Pipes to compile SCSS and minify assets. Please make sure you have the Hugo Extended version installed. If you are not using the extended version, some themes will not not compile.

Install Hugo

Hugo is available for a variety of operating systems and architectures. Visit the latest release page, and scroll down to the Assets section to download the version you need.

In this section, we take the Windows version for example.

  1. Download hugo_extended_0.111.3_windows-amd64.zip.
  2. Extract the archive.
  3. Move the executable to the desired directory, such as E:\hugo\bin.
  4. Add this directory to the PATH environment variable.
  • Type Environment Variables in the search box on the taskbar, and click on the Edit the system environment variables result.
  • Click the Environment Variables button under the Advanced tab.
  • In the pop-up window, select the Path variable under the System variables section and click Edit.
  • Click New in the Edit environment variable dialog, and input the path of the directory “E:\hugo\bin”.
  • Click **OK **to quit the dialogs in turn.

Check Hugo Version

Open the Command Prompt in Windows, and type hugo version to check if you have installed Hugo successfully.

Install Git

Git and Go are often used when working with Hugo.

Git is required to:

  • Use the Hugo Modules feature
  • Build Hugo from source
  • Install a theme as a Git submodule
  • Access commit information from a local Git repository
  • Host your site with services such as AWS Amplify, CloudCannon, Cloudflare Pages, GitHub Pages, GitLab Pages, and Netlify.

Go is required to:

  • Use the Hugo Modules feature
  • Build Hugo from source Please refer to the Git and Go documentation for installation instructions.
