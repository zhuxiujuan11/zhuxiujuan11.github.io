Deploy Hugo to GitHub Pages

As mentioned before, Hugo creates the entire static site in the public directory, so you only need to push the files under the public directory to the GitHub repository to get them published on GitHub Pages, here is how:

  1. When you finish the content editing in content/docs, run hugo to generate the latest Hugo site.
hugo

  1. Copy all files in the public directory to the local username.github.io directory created in the last section - Create a GitHub Page. The original index.html file can be replaced.

  2. Add, commit, and push your changes.

git add .
git commit -m "commit description" 
git push origin main
  1. Wait a few seconds for pages build and deployment, and view your GitHub Page. Your site is created!
