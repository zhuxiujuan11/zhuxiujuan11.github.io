Create a Hugo site

In the following steps, various command lines should be executed. Please notice that:

If you are a Windows user:

  • Do not use the Command Prompt.
  • Do not use Windows PowerShell.
  • Run these commands from PowerShell or a Linux terminal such as WSL or Git Bash.

PowerShell and Windows PowerShell are different applications.

Create your site

  1. Open a terminal, such as Git Bash.
  2. Switch to a local directory where you want to create the Hugo site.
  3. Run this command to create a fresh Hugo site in the folder mynewsite. The folder name is customizable.
hugo new site mynewsite

Install a theme

Choose a theme

  1. In Hugo Themes page, choose a theme you want.

  2. Install the theme and customize the website as the instructions on its page.

Install the theme

Take the Whisper theme under the Docs tag as an example.

Copy or git clone this theme into the themes folder mynewsite/themes

Install with Git

  1. Change the current directory to the root of your project.
cd mynewsite
  1. Initialize an empty Git repository in the current directory.
git init
  1. Clone the Ananke theme into the themes directory, adding it to your project as a Git submodule.
git clone https://github.com/zerostaticthemes/hugo-whisper-theme.git themes/hugo-whisper-theme

Install from .zip file

You can download the .zip file located here.

Extract the downloaded .zip inside the themes folder. Rename the extracted folder from hugo-whisper-theme-master -> hugo-whisper-theme. You should end up with the following folder structure mynewsite/themes/hugo-whisper-theme

Add your content

The fastest way to get started is to copy the example content of the theme, and modify the included config.toml

Copy exampleSite contents

Copy the entire contents of the exampleSite folder to the root folder of your Hugo site mynewsite.

Update config.toml

After you copy the config.toml into the root folder of your Hugo site, you will need to update the baseURL, themesDir and theme values in the config.toml as below.

baseURL = "/"
themesDir = "themes"
theme = "hugo-whisper-theme"

Add a new page

You can also create a new page on the site as below.

  1. In the root folder of your Hugo site, run this command.
hugo new docs/my-first-post.md

This theme generates documentation from markdown files located in content/docs. Do not put markdown files under other paths.

  1. Open my-first-post.md with your editor.
---
title: "My First Post"
date: 2023-05-10T09:03:20+08:00
draft: true
---

The draft value in the front matter is true. By default, Hugo does not publish draft content when you build the site. You can run hugo server -D to include draft content, or delete draft: true or change it into false to make the file published by running hugo server.

  1. Add some content to the markdown file.

Publish the site

When you publish your site, Hugo creates the entire static site in the public directory in the root of your project. This includes the HTML files, and assets such as images, CSS files, and JavaScript files.

In this step, you will publish your site in local, not deploy it.

After installing the theme for the first time, generate the Hugo site.

hugo
  1. For local development, run Hugo’s built-in local server.
hugo server

  1. Now enter localhost:1313 in the address bar of your browser to view the site locally.

  2. Keep the development server running as you continue to add and change content. You can see the changes in real time.
